Nowa wersja polskiego układu klawiatury, za pomocą której można otrzymać blisko 2000 symboli Unicode, układ maszynistki QWERTY.

W celu zainstalowania wybranego układu na komputerze konieczne jest pobranie pliku „setup.exe” i jego uruchomienie, a następnie ustawienie zainstalowanego układu w pasku zadań.

W przypadku, gdyby (np. na niektórych laptopach) instalacja była zablokowana wskutek komendy „Inna instalacja jest w toku”, należy włączyć menedżera zadań (Ctrl+Shift+Esc) i wyłączyć prawym przyciskiem myszy „Instalator systemu Windows”.
